Ubuntu 常用命令。。。


1 修改时区

```
sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
sudo dpkg-reconfigure locales
```

2 更改Locale

```
sudo vim /var/lib/locales/supported.d/local

sudo dpkg-reconfigure locales
```



3 同步网络时间
